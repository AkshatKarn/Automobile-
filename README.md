# 🚗 Automobile Production Delay Analysis & Prediction

## 📌 Project Overview
This project analyzes and predicts production delays in automobile manufacturing using data analysis and machine learning.

## 📊 Features
- Data Cleaning & Feature Engineering
- Exploratory Data Analysis (EDA)
- Delay Trend Analysis
- Machine Learning Model for Prediction

## 🧠 Tech Stack
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn

## 🤖 Model Used
- Linear Regression

## 📈 Key Insights
- Production delays increase with machine downtime and defects
- Supplier quality significantly impacts delay
- Certain production issues cause major bottlenecks

## 🚀 How to Run
1. Run `eda.py` → generates processed_data.csv  
2. Run `model.py` → trains model & predicts delay  

## 🔮 Sample Output
Predicted Production Delay: ~10 days
